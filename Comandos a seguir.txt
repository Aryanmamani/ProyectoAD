Descargar un repositorio
git clone "link del repositorio que se desea clonar"

Agregar un archivo
git add .

Modificar un archivo
git commit -m "Mensaje a escribir"
git push origin master

Comando para cambiar version
git tag 
git tag v0.0.1 -m "Mensaje"
git push origin master --tag
ó
git push origin v0.0.2
